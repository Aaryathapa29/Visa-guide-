# Frontend

This folder contains the Visa Guide React app.

The auth flow lives in [src/app/App.tsx](src/app/App.tsx) and is organized around these screens:

- Visa aspirant sign-up
- Visa aspirant sign-in
- Consultancy sign-up
- Consultancy sign-in

Run it from this folder with:

```bash
npm install
npm run dev
```

Do not commit `node_modules/`; it is generated by `npm install` and is ignored at the repo root.
